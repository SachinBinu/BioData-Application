# MERN Biodata Application

##### Overview
This is a full-stack MERN (MongoDB, Express, React, Node.js) application that allows users to register, log in, and submit their biodata including profile image. The application also includes role-based access where admins can view all users' biodata while normal users can only view their own.

##### Features
\- User Registration and Login (JWT Authentication)
\- Role-based Access (Admin / User)
\- Biodata Submission Form
\- Image Upload using Multer
\- User-specific Biodata View
\- Admin Dashboard to view all users

##### Tech Stack
Frontend:
\- React.js
\- Axios
\- React Router DOM

##### Backend:
\- Node.js
\- Express.js
\- MongoDB
\- Mongoose
\- JWT
\- Multer
\- Bcrypt

#### Setup Instructions

##### Backend:
cd server
npm install
npm start

##### Frontend:
cd client
npm install
npm start

##### Runs on:
Frontend: http://localhost:3000
Backend: http://localhost:5000
